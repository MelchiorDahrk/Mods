# No Resting and Waiting

***




## Description

A very simple Lua script that instantly closes the Rest box if opened without activating a bed. No options, no settings, no toggles. As long as this plugin is active, the Rest keybind is completely useless.

Why? Just a personal preference. I feel like 'waiting' shouldn't be an unlimited 'time-acceleration' superpower that one can trigger at will. And one shouldn't be able to recover from injuries by just lying down in the dust for a few hours.

- Works really well with [No Sneaky Sleeping](https://www.nexusmods.com/morrowind/mods/53539)
- If you want a less "radical", more feature-complete approach check out [Rest and Sleep Recovery Options](https://www.nexusmods.com/morrowind/mods/56604).
- If you want to be able to check time, I recommend using something like [TimeHUD - A Clock UI widget](https://www.nexusmods.com/morrowind/mods/57407).


## Installation

Extract to your Data Files/mods folder. Then add these lines in the correct spots in *openmw.cfg*:

```
data="/path/to/mod/NoRestingAndWaitingOpenMW"
content=No resting and waiting.omwscripts
```
